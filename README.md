# 🛒 TypeScript Shopping Cart

Simple shopping cart system built with **TypeScript** using **Object Oriented Programming (OOP)** concepts.

This project simulates a basic e-commerce cart where products can be added, quantities updated, and the final price calculated with discounts applied.

---

## 🚀 Features

* Product creation
* Category association
* Shopping cart system
* Add items to cart
* Automatic quantity update
* Discount calculation
* Total price calculation
* User roles using Enum (Admin / Customer)

---

## 🧠 Concepts Practiced

This project demonstrates the use of:

* TypeScript Classes
* Interfaces
* Enums
* Constructors
* Array methods (`map`, `reduce`, `some`)
* Object composition
* Basic project structure
